# 개인 포트폴리오 정적 웹사이트 설계서

**날짜**: 2026-04-19
**프로젝트**: 동기부여/기회제공 포트폴리오 사이트

## 목표

방문자에게 동기부여와 기회를 제공하고, 카카오톡 채널 상담으로 유도하는 개인 포트폴리오 웹사이트.

## 기술 스택

- 단일 HTML 파일 + Tailwind CSS CDN
- 빌드 도구 없음, 정적 호스팅 즉시 배포 가능

## 디자인 스타일

- 미니멀/모던, 화이트 기반
- 한국어 최적화 웹폰트 (Pretendard)
- 색상: 화이트 배경, 다크 텍스트, 1개 악센트 컬러

## 섹션 구성

### 1. Hero + About

- 전체 화면 높이의 히어로 영역
- 중앙에 이름 + 슬로건 ("동기부여와 기회를 제공합니다")
- 큰 타이포그래피, 미니멀 화이트 배경
- 스크롤 유도 화살표 애니메이션
- 자기소개 2-3줄 (가치/미션 강조)

### 2. Services (서비스/기회)

- 카드 레이아웃 3-4개 항목
- 각 카드: 아이콘 + 제목 + 짧은 설명
- 호버 시 그림자/이동 효과

### 3. Contact (카카오톡 상담 CTA)

- 강렬한 CTA 메시지: "지금 카카오톡으로 상담받기"
- 카카오톡 채널 링크 버튼
- 부가 메시지: "궁금한 점이 있으신가요? 카카오톡으로 편하게 문의하세요"

## 공통 요소

- 상단 고정 네비게이션 (투명 → 스크롤 시 화이트)
- 반응형: 모바일/태블릿/데스크톱 대응
- 스무스 스크롤

## 전환 목표

방문자 → 카카오톡 채널 상담 요청

## 파일 구조

```
index.html  (모든 콘텐츠가 포함된 단일 파일)
```
